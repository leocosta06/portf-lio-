# portfólio 
Leo Costa — Full Stack Developer & Software Engineering Student

Este repositório contém o meu portfólio profissional, desenvolvido para apresentar minhas competências técnicas, projetos em destaque e minha trajetória acadêmica no Centro Universitário Fametro.
Tecnologias Utilizadas

O projeto foi construído com foco em performance e estética moderna:

Frontend: HTML5 Semântico, CSS3 (Variáveis, Grid, Flexbox) e JavaScript Vanilla.

Animações: Intersection Observer API para efeitos de scroll e cursor personalizado com manipulação de DOM.

Design: Estética Dark Mode com foco em UI/UX para desenvolvedores.

Minha Stack Principal

Desenvolvo soluções robustas utilizando as seguintes tecnologias apresentadas no portfólio:

  Backend: Python (Flask/FastAPI), Node.js e Go.

  Frontend: React e JavaScript.

  QA & Testes: Conhecimento em automação com Cypress, Selenium e Postman.

  Ferramentas: Docker, Git, Linux e Bancos de Dados SQL.
